Surface Biology & Geology Visible to Shortwave Infrared Imaging Spectrometer

SBG VSWIR will leverage NASA's unique capabilities to spectrally image the planet every sixteen days. This webbook provides some basic content and navigation of the ongoing github based algorithm development. For more information, visit the mission website .

SBG VSWIR Overview

Specifications

First-of-its kind instrument

  • 16-day repeat orbit
  • 30m ground sampling distance
  • 400-2500 nm spectral range
  • 10 nm spectral sampling
  • 190 km swath
